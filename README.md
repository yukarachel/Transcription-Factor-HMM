Chromatin State Modeling using Histone Marks as part of Algorithms in Computational Genomics Course:

Implemented the Baum–Welch algorithm to train a Hidden Markov Model (HMM) for identifying and annotating promoter and gene body regioins in previously unannotated genomic sequences. Using promoter and gene interval data annotated with histone marks, I trained the HMM to learn underlying state transitions and emission probabilities, enabling prediction of likely gene bodies and promoter regions. This project involved extensive use of NumPy for efficient, vectorized computations and reinforced my ability to translate statistical models into scalable Python implementations.
